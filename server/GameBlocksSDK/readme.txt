Overview
========

This package includes the GameBlocks SDK, Code Examples and Documentation.

/Bin - Required GameBlocks dlls which must be placed in application dll path
/Config - Sample XML configuration files for development and production
/Doc - Index.html will invoke Integration guide and API reference. 
/Examples - Example solutions for VC9, VC10 and VC11 are available, which demonstrate connection to GameBlocks server and basic functionality. 
/Include - Header files required for GameBlocks integration
/Lib - Library files required for GameBlocks integration
readme.txt - This file.
